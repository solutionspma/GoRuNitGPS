Architecture Overview
	•	Frontend: HTML, CSS, JavaScript (mobile-first responsive design)
	•	Mapping: OpenRouteService API + native map integrations
	•	Voice: Web Speech API for navigation prompts
	•	Data Handling: Dynamic DOM-based route builder with extensibility for API persistence
	•	Integration Ready: Designed for CRM systems (Agent CRM, custom APIs)

Core Modules
